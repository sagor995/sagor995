I build immersive learning experiences — games that run inside 360° projection rooms, on VR headsets, and in students' browsers at the same time. Currently shipping XR curriculum tools at Wigan & Leigh College, where the work runs in actual lessons with actual students.

My background spans Unity, Unreal Engine, Flutter mobile apps, and web backends. MSc in Games & Extended Reality from the University of Salford (2024, Distinction). Before XR pulled me in completely, I spent three years in industry as a backend and mobile developer.

## 🚀 What I'm Currently Shipping

**Quiz Quest Suite** — five Unity XR educational games deployed in live FE curriculum. Runs on Igloo CAVE (360° projection), Meta Quest 3 (MR passthrough), and phone browsers simultaneously. Students join via QR code — no app install. Auto-generates PDF reports per player at session end.
